How to install a hidden socket under tile

Step-by-step flush mounting with WAVER STORE template — for kitchen, bathroom and large-format stoneware.

WAVER STORE hidden sockets mount flush with porcelain stoneware or tile — no plastic frame on the surface. This guide covers the typical sequence for electricians and tilers. Full PDF: download manual.

Tools and preparation

  • WAVER STORE kit with template
  • Tile cutter / angle grinder with diamond disc (for stoneware)
  • Standard electrical tools and flush box
  • Level and depth gauge for tile + adhesive thickness

Step-by-step installation

  1. Mark the cut-out — place the WAVER STORE template on the tile or wall, mark outline and depth.
  2. Cut the recess — cut porcelain stoneware to template size; deburr edges.
  3. Install the box — electrician sets flush box at correct depth for tile thickness (usually 6–20 mm cladding).
  4. Mount mechanism flush — frame and mechanism align with future tile plane.
  5. Tile and check — after cladding, verify cover sits in one plane with grout lines; test operation.

Tile thickness reference

CladdingTypical thicknessNote
Wall tile6–10 mmCheck total with adhesive
Porcelain stoneware8–12 mmLarge format — use template
Thick slab12–20 mmConfirm depth in order comment

Common mistakes

  • Box too shallow — mechanism protrudes past tile plane.
  • Cut-out oversize — gaps visible at corners.
  • Wrong zone in bathroom — always verify moisture zones first.

FAQ
